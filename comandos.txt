Endpoints

Usuarios
POST /api/usuarios - Crear un nuevo usuario OK
GET /api/usuarios - Listar todos los usuarios OK
GET /api/usuarios/:id - Obtener un usuario por su ID OK
PUT /api/usuarios/:id - Modificar un usuario por su ID OK
DELETE /api/usuarios/:id - Eliminar un usuario por su ID OK

Eventos
POST /api/eventos - Crear un nuevo evento OK
GET /api/eventos - Listar todos los eventos OK
GET /api/eventos/:id - Obtener un evento por su ID OK 
PUT /api/eventos/:id - Modificar un evento por su ID OK
PUT /api/eventos/:id/aprobar-rechazar - Aprobar o Rechazar un evento (se debe suministra unicamente revisionAprobado y mensajeAproba)
DELETE /api/eventos/:id - Eliminar un evento por su ID OK

Documentos Históricos
POST /api/doc-historicos - Crear un nuevo documento histórico OK 
GET /api/doc-historicos - Listar todos los documentos históricos OK
GET /api/doc-historicos/:id - Obtener un documento histórico por su ID OK
PUT /api/doc-historicos/:id - Modificar un documento histórico por su ID OK 
DELETE /api/doc-historicos/:id - Eliminar un documento histórico por su ID OK

Mapas
POST /api/mapas - Crear un nuevo mapa OK
GET /api/mapas - Listar todos los mapas OK
GET /api/mapas/:id - Obtener un mapa por su ID OK
PUT /api/mapas/:id - Modificar un mapa por su ID OK
DELETE /api/mapas/:id - Eliminar un mapa por su ID OK

Observaciones
POST /api/observaciones - Crear una nueva observación
GET /api/observaciones - Listar todas las observaciones
GET /api/observaciones/:id - Obtener una observación por su ID
PUT /api/observaciones/:id - Modificar una observación por su ID
PUT /api/observaciones/:id/responder - Responder observacion (se debe suministra unicamente revisionAprobado y mensajeAproba)
DELETE /api/observaciones/:id - Eliminar una observación por su ID
